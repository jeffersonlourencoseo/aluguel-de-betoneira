PASSO 1: Abrir o arquivo no GitHub

Clique no botao verde abaixo. Ele vai abrir o arquivo no GitHub pra voce editar.

PASSO 2: Achar o texto errado

Depois que abrir, aperte Ctrl + F no teclado.

Digite isso na caixinha que aparecer:

fullCanonical = `${siteUrl}${canonical}`;

O navegador vai achar essa linha sozinho e destacar em amarelo.

PASSO 3: Substituir pelo texto certo

Selecione (clique e arraste) por cima dessa linha destacada, aperte Delete no teclado.

Depois COLE isso no lugar (aperte Ctrl + V):

function normalizeCanonical(path: string): string { if (path === '/') return path; return path.endsWith('/') ? path : `${path}/`; } const fullCanonical = `${siteUrl}${normalizeCanonical(canonical)}`;

PASSO 4: Salvar

Role a pagina pra mbaixo ate o final.

Clique no botao verde escrito "Commit changes...".

PRONTO! O site atualiza sozinho em 2 minutos.
